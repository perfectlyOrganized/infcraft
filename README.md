# Infinity Craft (Forge 1.21.1)
### By perfectlyOrganized
Adds the concept of [Infinite Craft by Neal.Fun](https://neal.fun/infinite-craft/) to minecraft.

Heavily based of [Infinite Craft By Exclusivert](https://github.com/ExclusivertMods/Infinite-Craft-Java) (fabric 1.20.1)

Thank you Neal.Fun and Exclusivert for their work leading up to this.

## Pros:
- Infinite Crafting
- Works with mods
- Forge

## Cons:
- Needs a [Google AIstudio API key](https://aistudio.google.com/api-keys)
